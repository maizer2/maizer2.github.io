# UNIT 22) 의문사 be 주어

의문사란 궁금한 내용을 물어볼 때 사용하는 말.

* 의문사
    * who(누가)
    * when(언제)
    * where(어디서)
    * what(무엇을)
    * how(어떻게)
    * why(왜)

* 의문문의 종류
    * 의문사가 없는 것
        * yes/no로 간결하게 대답할 수 있다.
    * 의문사가 있는 것
        * 물어보는 것을 정확히 대답해야 한다.

* be동사 의문사의 어순
    * 의문사(who, when, etc.) + beV + S + ~?

# UNIT 23) 의문사 do 주어 + 동사원형

* 일반동사 의문사의 어순
    * 의문사 + do/does/did + S + V(동사원형)

의문사는 주어의 역할을 할 수 있다.

* 의문사 주어
    * S(의문사) + V + ~?

# UNIT 24) How ...?

* How의 용법
    * How + ~
        * 상황, 형편
            * How + beV + S + ...?
        * 방법
            * How + do/does/did + S + V(동사원형) + ...?
    * How + 형용사/부사 + ~
        * 정도
            * How + 형용사/부사 (+ 명사) + beV + S + ...?
                * how much
                * how old
                * how big
                * how long
                * how much money
                * how much time
                * how hard
                * how tall
                * how often
                * how fast
                * how many people
