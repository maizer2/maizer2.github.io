# UNIT 19) be동사의 부정문, 의문문

* be동사의 부정문
    * S + beV + not + ~
* be동사의 의문문
    * 질문) beV + S + ~
    * 대답) Yes/No, S(상대방의 주어) + beV(n't)

# UNIT 20) 일반동사의 부정문

* 일반 동사의 부정문
    * S + do/does/did + not + V(동사원형) + ~

# UNIT 21) 일반동사의 의문문

* 일반 동사의 의문문
    * 질문) Do/Does/Did + S + V(동사원형) + ~ ?
    * 대답) Yes/No + S(상대방의 주어) + do/does/did(n't).
