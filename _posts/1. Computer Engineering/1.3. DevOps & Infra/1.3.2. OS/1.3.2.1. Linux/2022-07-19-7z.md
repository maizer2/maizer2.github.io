```
sudo apt install p7zip p7zip-full

7za <command> [<switches>...] <archive_name> [<file_names>...] [<@listfiles...>]
```

### Example

#### Extract on current directory

7zr e "filename"

#### Extract on designated directory

7zr x "filename" "path"
